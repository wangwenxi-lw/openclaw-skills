# OpenClaw Skills Collection

本仓库包含 OpenClaw AI 助手常用的本地技能集合。

## 📦 技能列表

| 技能名称 | 说明 |
|----------|------|
| **agent-browser** | 浏览器自动化工具 |
| **find-skills** | 技能发现与安装 |
| **first-principles-skills** | 第一性原理学习系统 |
| **first-principles-study** | 第一性原理学习 |
| **github** | GitHub CLI 交互 |
| **notion** | Notion API 操作 |
| **obsidian** | Obsidian 笔记管理 |
| **openclaw-docs** | OpenClaw 文档参考 |
| **summarize** | URL/文件摘要工具 |
| **tavily-search** | Tavily AI 搜索 |
| **tencentcloud-lighthouse-skill** | 腾讯云轻量应用服务器 |
| **weather** | 天气查询 |

## 🚀 使用方法

将本仓库克隆到 OpenClaw 工作区的 skills 目录：

```bash
cd ~/.openclaw/workspace
git clone https://github.com/wangwenxi-lw/openclaw-skills.git skills
```

## 📝 自定义技能

每个技能目录包含：
- `SKILL.md` - 技能定义和使用说明
- 相关脚本和配置文件
